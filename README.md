# V-aipage
一个适用于openaiAPI格式的单页面无需任何构建的AI问答网页
拥有 __创建新对话__，__导出对话__，__修改参数__，__打断对话__，__复制对话__，__重新回答__ 基础功能，只有一个html文件，由Alpine.js和（AI）驱动捏。
### 只有一个HTML文件
__超轻量__ 适用于简单的llama部署后简单的前端问答功能
